# RS Assistant v100

AI Voice Assistant for Android with complete phone control.

## Features
- Voice Recognition (Hindi/English/Hinglish)
- Wake Word Detection
- Phone Control via Accessibility Service
- Camera Control
- Hand Gesture Detection
- OAuth Authentication via chat.z.ai
- Background Voice Service

## Voice Commands
- "Call [contact]" - Make phone calls
- "Message [contact]" - Send messages
- "Open [app]" - Launch applications
- "Flashlight on/off" - Control torch
- "WiFi settings" - Open settings
- "What time is it" - Get current time
- "Scroll up/down" - Navigation gestures
- And many more...

## Requirements
- Android 7.0 (API 24) or higher
- Permissions: Microphone, Camera, Phone, Contacts, SMS
