# RS Assistant v101

Complete AI Voice Assistant for Android with always-running background service.

## Features
- Voice Recognition (Hindi/English/Hinglish)
- Wake Word Detection ("Hey Assistant")
- Always-On Background Service
- Phone Control via Accessibility Service
- Camera & Flashlight Control
- Hand Gesture Detection
- Face Detection
- Location Tracking
- OAuth Authentication via chat.z.ai
- Notification Listener
- Auto-restart on boot

## Voice Commands
- "Call [contact]" - Make phone calls
- "Message [contact]" - Send messages
- "Open [app]" - Launch applications
- "Flashlight on/off" - Control torch
- "WiFi/Bluetooth settings" - Open settings
- "What time is it" - Get current time
- "Scroll up/down" - Navigation gestures
- "Take screenshot" - Capture screen
- "Search [query]" - Web search
- "YouTube [query]" - YouTube search
- And many more...

## Requirements
- Android 7.0 (API 24) or higher
- Grant all permissions for full functionality
- Enable Accessibility Service for phone control
- Disable battery optimization for always-on service
